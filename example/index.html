Phoenix-styles - Style Library for Mobile App

使用方式：

  1. 1、简单粗暴直接引入:

    git拉下来代码，直接项目引入css: https://github.com/future-team/phoenix-styles

  2. 2、通过bower安装:

            $ sudo npm install bower -g
        $ cd phoenix-styles
        $ bower install phoenix-styles --save

  3. 3、通过git submodule依赖:

            * 如果你只需要某一部分的样式，这样我们还是支持的：
        //首先通过git submodule的方式引入phoenix-styles
        $ git submodule add https://github.com/future-team/phoenix-styles.git
        然后，如果想只引入部分less，你可以这样：
        @import '../../phoenix-styles/less/public.less';
结构目录：

  1. - dist/ 生成文件目录

    - fonts/ 字体文件(eot/svg/ttf/woff)
    - phoenix-styles.css/phoenix-styles.min.css 主样式文件，阿波罗主题
    - ios-skin.css/ios-skin.min.css IOS主题样式文件

  2. - less/ 不同模块的样式文件：如果只需要引入某些文件可以单独引入。

    - common/ 公共文件，包含重置文件和全局共享类
    - fonts/ 字体文件(eot/svg/ttf/woff)
    - mixins/ 对应每个模块的样式文件
    - modules/ 对应每个模块的样式文件
    - phoenix-styles.less 引入所有相关文件，flex布局
    - phoenix-styles-grid.less 引入所有相关文件，float布局
    - skin.less 默认皮肤(阿波罗)
    - ios-skin.less IOS皮肤
    - variable.less 定义变量，例如颜色变量等全局变量
    - public.less 引入了所有公共方法的less
    - mixins.less 引入mixins下所有文件
    - fonts.less 字体样式

  3. - example/ 文档和示例

    － src/ 示例源文件
    － index.html 入口文件
    － doc.html 详细文档
Let`s get start!
